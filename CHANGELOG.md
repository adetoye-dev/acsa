All notable changes to Acsa should be documented in this file.

## Unreleased

## 0.1.0

Initial public release foundation:

- Rust workflow engine with DAG execution and SQLite persistence
- YAML workflow-as-code model
- built-in trigger, logic, integration, AI, and human-gate nodes
- subprocess and WASM connector runtime
- Next.js visual editor with run history and human-task inbox
- observability endpoints for metrics, logs, and run history
- Docker, Kubernetes, and release packaging assets
